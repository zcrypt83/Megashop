# Megashop Frontend

SPA construida con React, React Router, Bootstrap, Lucide y Vite.

## Ejecutar

```powershell
npm.cmd install
npm.cmd run dev
```

## Compilar

```powershell
npm.cmd run build
```

La API usa `VITE_API_URL` o, de forma predeterminada:

```text
http://localhost:4000/api/v1
```

Las pantallas actuales incluyen catalogo, carrito visual, seguimiento, login y dashboard administrativo.
